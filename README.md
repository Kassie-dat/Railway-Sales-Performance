# Railway-Sales-Performance
Data analysis project using Excel and Power BI to uncover business insights through interactive dashboards.
Project Overview

This project analyzes railway ticket sales data to understand revenue performance, passenger behavior, ticket purchasing patterns, and route demand. The goal was to identify key trends that can help railway operators optimize services, improve customer experience, and increase revenue.

Tools Used:

Microsoft Excel

Power BI

Power Query

DAX

Problem Statement


Railway operators generate large volumes of ticket sales data but often lack clear visibility into revenue trends, customer purchasing behavior, and route performance. This project was conducted to analyze railway sales data and provide actionable insights for operational and strategic decision-making.

Data Source

Historical railway ticket sales dataset


Key Insights

Identified the routes generating the highest revenue.

Analyzed customer preferences for ticket types and travel classes.

Determined peak booking and travel periods.

Evaluated payment method usage patterns.

Highlighted opportunities for improving service efficiency and revenue generation.
